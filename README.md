# 🚀 About This Repository

This repository contains multiple Exploratory Data Analysis (EDA) projects performed on datasets from different domains such as healthcare, education, aviation, insurance, employee analytics, tourism, and app analytics.

These projects focus on:
- Data Cleaning & Preprocessing
- Statistical Analysis
- Exploratory Data Analysis (EDA)
- Data Visualization
- Trend & Pattern Discovery
- Business Insight Generation

All datasets used in these projects are sourced from Kaggle.
